# F1 Blacklist — калькулятор шансов на титул

Простое одностраничное приложение (HTML / CSS / JS) в стилистике «Blacklist» из *Need for Speed: Most Wanted*: календарь **Формулы-1 2026** на **22 этапа** (без отменённых весной Гран При), **22 пилота**, учёт **спринт-уикендов** при расчёте максимума очков.

## Возможности

- Выбор числа **уже прошедших** этапов — остаток сезона и список оставшихся ГП (с пометкой спринтов).
- Ручной ввод **очков** по каждому пилоту; список **сортируется по убыванию** очков.
- **Математика:** остаётся ли шанс на титул (сравнение с лидером и суммой максимумов на оставшихся этапах).
- **Индекс %:** насколько максимально возможный итог пилота (если на каждом оставшемся ГП он берёт максимум по модели) близок к такому же «потолку» у лидера. Очки за лучший круг **не** учитываются.
- Сохранение **очков** и **числа этапов** в `localStorage` браузера.
- Приветственное окно с кнопками «Ясно» и «Больше не показывать».

## Запуск локально

Откройте [`index.html`](index.html) в браузере (двойной клик или через локальный сервер — не обязательно).

## Публикация на GitHub и GitHub Pages

Репозиторий уже с инициализированным Git (ветка `main`). Дальше — с вашей машины.

### 1. Войти в GitHub CLI

```bash
gh auth login
```

Выберите GitHub.com → HTTPS → авторизация через браузер.

### 2. Создать репозиторий и отправить код

В каталоге проекта:

```bash
cd "путь/к/f1-blacklist"
gh repo create f1-blacklist --public --source=. --remote=origin --push
```

Если репозиторий уже создан вручную на github.com:

```bash
git remote add origin https://github.com/<ВАШ-НИК>/f1-blacklist.git
git push -u origin main
```

### 3. Включить GitHub Pages

**Через сайт:** репозиторий → **Settings** → **Pages** → **Build and deployment** → **Source: Deploy from a branch** → ветка **main**, папка **/ (root)** → **Save**.

Через 1–2 минуты сайт появится по адресу:

**`https://<ВАШ-НИК>.github.io/f1-blacklist/`**

(Если назвали репозиторий иначе — замените последний сегмент в URL.)
